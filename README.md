#regtools

Tools that integrate DNA-seq and RNA-seq data to help interpret mutations
in a regulatory and splicing context.

##Usage:

```
    regtools --help
```

##Installation

Clone and install regtools by running:
```
    git clone https://github.com/griffithlab/regtools
    cd regtools/
    mkdir build
    cd build/
    cmake ..
    make
```
